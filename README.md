🌏 Atulya Rangmanch – UNESCO Heritage Explorer
A visually engaging, single-page website showcasing some of India’s most iconic UNESCO World Heritage Sites. The project is designed to provide users with a quick, aesthetic, and informative exploration experience, combining cultural storytelling with modern web design.
🚀Features
🎯 Single-page scroll website with smooth navigation
🖼️ Interactive card-based layout for major heritage sites
🔗 Clickable cards leading to detailed individual pages
🎨 Clean UI with animations and hover effects
📱 Responsive and visually appealing design
🏛️ Covered Heritage Sites
Sun Temple Konark
Sundarbans National Park
Great Living Chola Temples
Hampi
Hill Forts of Rajasthan
Churches and Convents of Goa
🛠️ Technologies Used
HTML5
CSS3
JavaScript
